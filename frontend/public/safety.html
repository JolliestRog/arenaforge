Safety

How DeckForge keeps your data safe.

No accounts, no stored data

DeckForge has no user accounts and stores nothing about you. Your collection and deck choices exist only in your browser tab. Closing the tab discards everything.

The exporter tool

The optional Windows exporter reads your MTG Arena installation locally on your PC to produce a collection file. It does not send data over the network — it writes a plain text file to your hard drive that you then paste into DeckForge. The source code is available on GitHub for inspection.

Windows SmartScreen warning

When you first run the exporter .exe, Windows may show a SmartScreen warning. This is normal for open-source tools distributed without a commercial code-signing certificate ($400/yr). Click More info → Run anyway. If you'd prefer not to do that, you can build the tool yourself from the source code.

Reporting a security issue

If you discover a vulnerability or something that looks wrong, email chandler@facey.page. Please don't publish it publicly before giving a chance to respond.
